# Gemini AI 프롬프트 설계 문서

## 기능 설명

Gemini API를 사용하여 사용자의 운동 기록 및 식단 기록을 분석하고 AI 기반 피드백을 제공한다.

AI 코치는 사용자의 운동량, 식단 균형 및 목표 체중 정보를 기반으로 운동 및 식단 개선 방향을 제안한다.

---

# AI 코치 기능 목표

1. 운동량 분석
2. 식단 균형 분석
3. 단백질 섭취량 분석
4. 운동 강도 분석
5. 개선 방향 제안
6. 사용자 맞춤형 피드백 제공

---

# 입력 데이터

AI 분석에 사용되는 데이터는 다음과 같다.

| 데이터   | 설명                   |
| ----- | -------------------- |
| 키     | 사용자 키                |
| 몸무게   | 현재 몸무게               |
| 목표 체중 | 목표 체중                |
| 운동 기록 | 운동명, 세트 수, 반복 횟수, 무게 |
| 식단 기록 | 칼로리, 탄수화물, 단백질, 지방   |

---

# 예상 분석 항목

## 운동 분석

* 운동량 적절성
* 운동 강도
* 운동 균형
* 운동 부족 여부

---

## 식단 분석

* 총 칼로리 적절성
* 탄수화물 비율
* 단백질 섭취량
* 지방 섭취량

---

## 개선 방향 분석

* 부족한 영양소 추천
* 운동 강도 조절 추천
* 식단 개선 방향 추천

---

# 기본 프롬프트 예시

```text id="lgjlwm"
너는 헬스 초보자를 위한 운동 및 식단 코치야.

사용자 정보:
- 키: 175cm
- 몸무게: 70kg
- 목표 체중: 68kg

오늘 운동 기록:
- 벤치프레스 3세트 10회 60kg
- 스쿼트 4세트 12회 80kg

오늘 식단 기록:
- 총 칼로리: 1850kcal
- 탄수화물: 120g
- 단백질: 90g
- 지방: 32g

위 정보를 바탕으로 아래 항목을 평가해줘.

1. 운동량 평가
2. 식단 균형 평가
3. 개선점 제안

답변은 짧고 이해하기 쉽게 작성해줘.
```

# 예상 출력 예시

```text id="9nl8hv"
운동량은 적절한 편입니다.
하체 운동 비중이 높아 균형이 좋습니다.

단백질 섭취량은 양호하지만,
탄수화물 섭취가 약간 부족한 편입니다.

운동 후 탄수화물 섭취를 조금 늘리는 것을 추천합니다.
```

# AI 응답 형식 목표

AI 응답은 아래 조건을 만족해야 한다.

1. 짧고 이해하기 쉬운 문장 사용
2. 운동 초보자도 이해 가능한 표현 사용
3. 과도하게 전문적인 용어 사용 금지
4. 긍정적인 피드백 포함
5. 개선 방향 제안 포함
